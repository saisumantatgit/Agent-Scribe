# Agent-Scribe — Governance Commands

## Session Start
At the beginning of each session, check `docs/logbook/` for the most recent entry and read its Handoff Notes section.

## /logbook
At session end, create a logbook entry. See `prompts/logbook.md` for instructions and `templates/logbook-template.md` for structure.

## /draft-aar
After completing a milestone, create an After Action Review. See `prompts/draft-aar.md` for instructions and `templates/aar-template.md` for structure.

## /draft-pir
After an incident, create a blameless Post-Incident Review. See `prompts/draft-pir.md` for instructions and `templates/pir-template.md` for structure.

## /draft-adr
For significant architectural decisions, create a Decision Record. See `prompts/draft-adr.md` for instructions and `templates/adr-template-madr-v4.md` for structure.
