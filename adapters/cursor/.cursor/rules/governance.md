# Agent-Scribe Governance Rules

When the user asks to write a logbook entry, follow the instructions in `prompts/logbook.md` using the template in `templates/logbook-template.md`.

When the user asks to draft an AAR, follow `prompts/draft-aar.md` using `templates/aar-template.md`.

When the user asks to draft a PIR, follow `prompts/draft-pir.md` using `templates/pir-template.md`.

When the user asks to draft an ADR, follow `prompts/draft-adr.md` using `templates/adr-template-madr-v4.md`.

At session start, check `docs/logbook/` for the most recent entry and load its Handoff Notes.
